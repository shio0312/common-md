# Senior QA 検証チェックリスト

## 機能・UI 検証
- [ ] 正常系フロー全通り動作確認（ブラウザ実操作）
- [ ] エラー系フロー（API エラー・空データ・権限なし・タイムアウト）
- [ ] ローディング状態の表示
- [ ] 空状態（データなし）の表示
- [ ] フォームバリデーション（必須項目・形式・文字数制限）
- [ ] 非同期操作中の二重送信防止

## 技術品質
- [ ] Console エラーなし（devtools → Console タブ）
- [ ] Network エラーなし（devtools → Network タブ）
- [ ] TypeScript 型エラーなし（`npx tsc --noEmit`）
- [ ] Lint エラーなし
- [ ] 未使用の import・変数なし

## レスポンシブ
- [ ] モバイル（375px）表示確認
- [ ] タブレット（768px）表示確認
- [ ] デスクトップ（1280px）表示確認
- [ ] タッチターゲット 44×44px 以上

## アクセシビリティ（WCAG AA）
- [ ] Tab キーでフォーカス移動できる
- [ ] フォーカスリング（アウトライン）が見える
- [ ] 色だけで情報を伝えていない
- [ ] テキストのコントラスト比 4.5:1 以上
- [ ] 大きなテキスト（18pt 以上）のコントラスト比 3:1 以上
- [ ] `<img>` に alt テキストがある（装飾画像は `alt=""`）
- [ ] フォームラベルが input と紐付いている（`htmlFor` / `aria-labelledby`）
- [ ] エラーメッセージがスクリーンリーダーに伝わる（`role="alert"` / `aria-live`）
- [ ] ダイアログに `role="dialog"` と `aria-labelledby` がある
- [ ] アイコンのみのボタンに `aria-label` がある

## パフォーマンス
- [ ] 不要な re-render がない（React DevTools で確認）
- [ ] API 呼び出しが重複していない
- [ ] 画像が最適化済み（Next.js `<Image>` コンポーネント使用）
